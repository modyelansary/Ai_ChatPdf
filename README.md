# 🤖 AI PDF Assistant

A local RAG application that allows chatting with any PDF using:

- LangChain
- ChromaDB
- Ollama
- Streamlit
- HuggingFace Embeddings

## Features

- 📄 Upload any PDF
- 💬 Ask questions
- 📚 Summarize documents
- 🌍 Arabic & English support
- 🧠 Local LLM (Llama 3.2)

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```
